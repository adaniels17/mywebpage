Course Summary

BUS 118W is Web Based Computing and it is about the programming for web based applications. Web Based computing is a technology that enables a person to move his applications from their storage to the internet. We will use HTML, CSS, and JavaScript and be implementing these into our websites. The course teaches you how to design your own website and create other web applications and by the end we will have a fully functional website which will be our final project. We will learn how to launch and test our site using different softwares.

    Lecture Review

    The lecture is an overview of HTTP and concepts. It gives backstory on how hypertext and the innovation of HTTP requests and other things like URLs. There are also examples of web pages written in HTML and requests. They also explain the concepts of cookies and how they're used. Proxy Server and their caches are explained and a diagram is showed of a proxy server and how they connect clients to a server.
